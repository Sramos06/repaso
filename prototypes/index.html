Reviewer Hub · Prototype Round 1 · July 9, 2026

Three directions.
Pick one — or franken-mix them.

Each one is the same app — library, upload, viewer, notes — wearing a completely different personality. All three are fully clickable: open a reviewer, toggle the notes panel, type in it (notes even persist). Each carries a candidate app name so you can pick a vibe and a name in one go.

mix & match welcome — e.g. "B's layout, C's typography, keep the name Repaso"
DIRECTION ARepaso

Warm paper desk. Cream + ink + terracotta, editorial serif, your reviewers as tactile index cards with washi tape. Feels like a cozy, organized study desk — handmade, personal, calm in daylight.

DIRECTION BAral

Night library. Deep ink + lamplight amber, refined serif, reviewers shelved as book spines in a sidebar layout. Calm and quiet for late-night sessions — dark done with restraint, zero gamer energy.

Open full prototype →
DIRECTION CSTUDYO

Swiss utility. Off-white, hard black rules, one electric blue, expanded grotesque type, numbered index rows that invert on hover. A precision tool — sharp, fast, zero decoration that doesn't work for a living.
